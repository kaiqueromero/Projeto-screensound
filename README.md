### English: 👇🏻 <br>
Academic Project focused on Java Learning; 
This project is a simulation of a playlist using Postgres as the database, where you can add songs, artists, etc. 

#### To run it, simply have a Java IDE like IntelliJ, the JDK installed, and the Postgres database. The JDK version used in the project is 22. Once that’s done, clone the repository and open it in your IDE; 
#### Don’t forget to configure Postgres in the file "/resources/application.properties".

### Portuguese: 👇🏻 <br>
Projeto Academico com foco no Aprendizado de Java;
Esse projeto é a simulação de uma playlist utilizando como banco de dados o postgres, onde você pode adicionar musicas artistas e etc..

#### Para executá-lo, basta ter uma IDE Java como IntelliJ, a JDK instalada e o banco de dados Postgres. A versão da JDK utilizada no projeto é a 22. Feito isso, clone o repositório e abra-o na sua IDE;
